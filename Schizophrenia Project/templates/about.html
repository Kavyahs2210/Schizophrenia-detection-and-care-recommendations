About This System

Purpose

This Schizophrenia Detection System is an educational tool designed to demonstrate the application of machine learning in mental health assessment. It uses multiple ML algorithms to analyze behavioral and physical indicators that may be associated with schizophrenia.

Technology Stack

Python & Flask

Backend framework

TensorFlow

Neural network model

Scikit-learn

ML algorithms (SVM, KNN, RF)

REST API

Programmatic access

Available Models

Random Forest (RF)

Ensemble learning method that operates by constructing multiple decision trees. Recommended for balanced accuracy and interpretability.

Support Vector Machine (SVM)

Powerful classifier that finds optimal hyperplane for separating classes. Effective for high-dimensional data.

K-Nearest Neighbors (KNN)

Instance-based learning algorithm that classifies based on similarity to training examples.

Neural Network (NN)

Deep learning model that can capture complex non-linear relationships in data.

Input Features

  • Age: Patient's age (0-120 years)
  • Gender: Male or Female
  • Marital Status: Single or Married
  • Fatigue: Level of tiredness (0-10 scale)
  • Psychomotor Slowing: Reduced speed of thought and movement (0-10)
  • Pain: Physical pain level (0-10)
  • Hygiene: Self-care and hygiene issues (0-10)
  • Movement: Abnormal movements or motor symptoms (0-10)

API Usage

The system provides REST API endpoints for programmatic access:

Single Prediction

POST /api/predict
Content-Type: application/json

{
  "features": {
    "Age": 35,
    "Gender": 0,
    "Marital_Status": 1,
    "Fatigue": 5.5,
    "Slowing": 3.2,
    "Pain": 2.1,
    "Hygiene": 1.5,
    "Movement": 2.8
  },
  "model_type": "rf"
}

Compare All Models

POST /api/compare
Content-Type: application/json

{
  "features": {
    "Age": 35,
    "Gender": 0,
    ...
  }
}
